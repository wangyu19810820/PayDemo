ABOUT

BitviseSSHClientPortable is a registry encapsulation program and launcher for the powerful and feature-rich SSH client BitviseSSHClient (from www.bitvise.com). The launcher is designed for use with PortableApps Suite (see http://www.portableapps.com) but will work on any portable media (or even fixed disks, for that matter).

BitviseSSHClient includes, among other tools, SFTP and Terminal clients.

Included in this distribution is BitviseSSHClient v4.51.

Learn more about BitviseSSHClient at http://www.bitvise.com.

USE

Run BitviseSSHClientPortable.exe from the BitviseSSHClientPortable directory.

Some settings for running BitviseSSHClientPortable, as well as passing parameters to underlying BitviseSSHClient application, are managed via an INI file. Copy the sample BitviseSSHClientPortable.INI file from BitviseSSHClientPortable\Other\Source directory into the BitviseSSHClientPortable directory in order to use the INI settings. The documentation in the INI file explains the options available.

It is recommended that BitviseSSHClient Profiles are loaded from / saved to your portable drive.

 

PORTABILITY NOTE / KNOWN BUGS
